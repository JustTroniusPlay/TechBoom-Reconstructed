# TechBoom-Reconstructed
Minecraft Modpack main scripts and configs.

TechBoom - technological modpack with main idea of exploration of various tech mods and using them to find creative solutions for different problems. But mostly to have fun!

The main goal is to balance things out. I am trying to achieve the same feel as The Lost Era modpack, where mods are intertwined, but you can take different paths as you desire.

Additionally there some exploration and combat aspects to not get bored by just constantly sitting on your base!
